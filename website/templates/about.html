TThe project's data set originates from Football Players Data on Kaggle Datasets. This comprehensive dataset offers detailed information on approximately 17,000 FIFA football players, meticulously scraped from SoFIFA.com.

The original data set has 17,000 rows and 51 columns. For the convenience of display, we selected 8 columns to display, which are Name, Age, Birthdate, Height (cm), Weight (kg), Positions, Nationality, Overall Rating, and 500 rows.

Features Description Data Type
Name Name of the player String
Age Age of the player Integer
Birthday Date of birth of the player Date
Height (cm) Player's height in Centimeters Float
Weight (kg) Player's weight in Kilograms Float
Position Positions the player can play String
Nationality Player's nationality String
Overall Rating Overall rating of the player in FIFA Integer
